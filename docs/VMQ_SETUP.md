# V免签支付系统配置指南

## 简介

V免签是一个开源的个人免签收款解决方案，无需申请商户，资金直接到账个人微信/支付宝账户。

## 工作原理

1. 用户下单时，系统会在商品原价基础上添加随机金额（如10.23元）
2. 用户扫码支付时，必须支付完全一致的金额
3. 监控端APP监听到收款通知后，将金额信息发送给系统
4. 系统根据金额匹配订单，自动完成发卡

## 配置步骤

### 1. 数据库设置

在Supabase SQL编辑器中执行 `supabase/migrations/vmq_setup.sql` 文件中的SQL语句，创建必要的表和字段。

### 2. 后台配置

1. 登录后台管理系统
2. 进入"系统设置" → "V免签支付设置"
3. 启用V免签支付
4. 启用需要的支付方式（微信/支付宝）
5. 设置通信密钥（用于验证监控端通知）
6. 复制"监控端配置地址"

### 3. 添加收款码

1. 进入后台"V免签管理"页面
2. 点击"添加收款码"
3. 填写收款码名称、选择支付类型
4. 上传收款码图片到图床，填写图片URL
5. 保存

### 4. 配置监控端APP

1. 准备一部安卓手机
2. 下载V免签监控端APK（开源版本）
3. 安装并打开APP
4. 在设置中填写回调URL（从后台复制的"监控端配置地址"）
5. 开启辅助服务，开始监听收款通知

## 监控端配置

监控端APP需要配置以下参数：

- **回调URL**: `https://your-project.supabase.co/functions/v1/vmq-notify`
- **通信密钥**: 与后台设置的密钥一致

## 注意事项

1. **手机保持在线**: 监控端手机需要保持网络畅通
2. **后台白名单**: 将微信/支付宝和监控端APP添加到后台白名单，防止被系统清理
3. **通知权限**: 确保监控端APP有通知栏访问权限
4. **金额匹配**: 用户支付金额必须与订单金额完全一致，否则无法匹配
5. **订单有效期**: 订单有效期为10分钟，过期后无法支付

## 常见问题

### Q: 用户支付后没有自动发卡？

A: 请检查：

1. 监控端APP是否正常运行
2. 回调URL是否正确配置
3. 通信密钥是否一致
4. 用户支付金额是否与订单金额完全一致

### Q: 如何测试是否配置成功？

A: 可以在后台添加测试用的收款码，然后下一笔测试订单，查看监控端是否收到通知。

### Q: 支持哪些收款方式？

A: 支持微信个人收款码和支付宝个人收款码。

### Q: 资金安全吗？

A: 资金直接进入您的个人微信/支付宝账户，不经过任何第三方平台。

## 技术说明

### Edge Functions

- `vmq-create-order`: 创建V免签订单
- `vmq-notify`: 接收监控端支付通知
- `vmq-query-order`: 查询订单状态

### 数据库表

- `vmq_qrcodes`: 存储收款码信息
- `orders`: 订单表（已添加pay\_amount、expire\_time等字段）

详见 `src/api/vmqApi.js` 文件。
